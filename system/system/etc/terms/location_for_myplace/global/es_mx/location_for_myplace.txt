Uso de la información de ubicación actual para Mis lugares

Este servicio Mis lugares ("Servicio") utiliza la información de ubicación del dispositivo del usuario para analizar el patrón de las ubicaciones en las que el usuario ha permanecido durante un tiempo prolongado, y para proporcionar la función de reconocer automáticamente dicha ubicación o permitir al usuario designar manualmente una ubicación, como "Mis lugares" (p. ej., casa, lugar de trabajo). Seguir con el análisis del patrón puede cambiar automáticamente la ubicación de una ubicación designada previamente. 
Aunque este Servicio utiliza información de ubicación (estación transceptora base (BTS), GPS, Wi-Fi, posiciones de red (NLP)) almacenada en el dispositivo del usuario, LG Electronics Inc. (LGE) nunca recopila o usa información de ubicación del usuario y solo utiliza tiendas en el dispositivo del usuario.
Este Servicio comienza a reconocer automáticamente los lugares donde el usuario ha permanecido durante un tiempo prolongado, comenzando cuando el usuario acepta el uso de la información de ubicación actual del dispositivo del usuario, y continúa a menos que el usuario deje de usar el Servicio apagando la correspondiente opción de función.
Sin embargo, LGE solo usará la información de ubicación actual del usuario si el usuario activa la opción de usar el servicio de ubicación de Google en el dispositivo.
