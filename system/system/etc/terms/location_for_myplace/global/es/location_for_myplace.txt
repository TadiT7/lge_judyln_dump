Uso de la información de ubicación actual para Mis lugares

Este servicio Mis lugares (en adelante denominado el "Servicio") utiliza la información de ubicación del dispositivo del usuario para analizar el patrón de las ubicaciones en las cuales el usuario ha estado durante un tiempo prolongado y para proporcionar la función para reconocer automáticamente la ubicación mencionada o permitir al usuario designar manualmente una ubicación, como "Mis lugares" (por ejemplo, casa, lugar de trabajo). El análisis continuo de patrones puede cambiar automáticamente la ubicación de una ubicación previamente designada. 
Aunque este Servicio utiliza información de ubicación (estación transmisora base (BTS), GPS, Wi-Fi, posiciones de red (NLP)) almacenada en el dispositivo del usuario, LG Electronics Inc. (LGE) nunca recopila ni usa información de ubicación del usuario y solo utiliza y almacena en el dispositivo de usuario.
Este Servicio comienza reconociendo automáticamente los lugares en los cuales el usuario ha estado durante un tiempo prolongado, comenzando cuando el usuario acepta el uso de la información de ubicación actual del dispositivo del usuario y continúa a menos que el usuario deje de usar el Servicio apagando la correspondiente opción de función.
No obstante, LGE solo usará la información de ubicación actual del usuario si el usuario activa la opción para usar el servicio de ubicación de Google del dispositivo..
