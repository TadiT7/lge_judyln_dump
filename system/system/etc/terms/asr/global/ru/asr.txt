Согласие на использование голосовой информации для автоматического распознавания речи Google («ASR») 

Данная услуга Parking Assistant («Услуга») использует функцию Google ASR, предоставляемую Google, Inc. («Google»), которая позволяет нам преобразовывать голос пользователя в текст.
Голосовая информация для использования данной Услуги передается и хранится на Google. Google будет использовать эту информацию в целях преобразования вашего голосового сообщения в текст и передачи этого текста LGE.
Вся собранная с помощью ASR информация регулируется политикой конфиденциальности Google, с которой можно ознакомиться на (https://www.google.com/policies/privacy/).
Вы также можете выключить ASR, отказавшись от услуг в «Меню "Параметры" - Юридическая информация».
LGE никогда не собирает и/или не использует какую-либо личную информацию, включая голосовую информацию пользователя, применяя для данной Услуги только ту информацию, которая не предназначена для личной идентификации, например, тексты.
Более того, LGE никогда не использует (и не позволяет третьим сторонам) текстовую информацию для отслеживания или сбора какой-либо личной информации пользователей Услуги и не ассоциирует собранные для этой Услуги данные с личной информацией из любого источника.
