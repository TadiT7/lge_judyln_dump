2.	Übersicht über FOTA-Softwareaktualisierungen
Von Zeit zu Zeit kann es erforderlich sein, die LGE-Firmware auf Ihrem Mobiltelefon (die „Software“) zu aktualisieren oder es müssen Ergänzungen vorgenommen werden, die von LGE bereitgestellt werden, oder für die LGE möglicherweise Software-Zusatzkomponenten bereitstellt. Das Durchführen einer Softwareaktualisierung führt dazu, dass neue Software auf Ihrem Mobiltelefon installiert wird, um diese Aktualisierungen, Ergänzungen oder Zusatzkomponenten bereitstellen zu können. Mit Softwareaktualisierungen bekommen sie die Möglichkeit, neue Eigenschaften oder Funktionen hinzuzufügen und in bestimmten Fällen auch vorhandene Eigenschaften oder Funktionen zu entfernen. Nach dem Fertigstellen der Softwareaktualisierung können Sie möglicherweise nicht mehr auf die zuvor installierte Version der Software zurückgreifen. 

Da die Softwareaktualisierung über einen FOTA-Vorgang durchgeführt wird, können, abhängig von Ihrem Netzbetreiber und Ihrem Tarif, zusätzliche Kosten anfallen, wann immer Sie neue Software über ein Mobilfunknetz herunterladen. Falls Sie eine Verwendung von Mobilfunkdaten für Softwareaktualisierungen vermeiden möchten, wählen Sie unter Einstellungen die Option „Nur Wi-Fi“ aus. 

a.	Ihre Optionen für die Verwaltung künftiger Softwareaktualisierungen
Sollten Sie in diese FOTA-Datenschutzrichtlinie einwilligen, sucht Ihr Mobiltelefon von Zeit zu Zeit nach neuer oder aktualisierter Software und installiert diese neue oder aktualisierte Software automatisch, wann immer diese verfügbar ist (Funktion „Automatisch prüfen“). Nach dem Installieren der Software benachrichtigen wir Sie mithilfe eines Popup-Hinweises oder mittels einer Benachrichtigung auf Ihrem Mobiltelefon. Sie erhalten die Möglichkeit, die Softwareaktualisierung vor dem Anwenden der Software zu akzeptieren oder abzulehnen. Die Software wird von Ihrem Mobiltelefon, welches diese automatisch um Mitternacht installiert, angewendet (Funktion „Automatische Aktualisierungen). Sie können die Funktionen „Automatisch prüfen“ und „Automatische Aktualisierungen“ unter [Einstellungen – System – Update-Center – Software Update – Overflow-Menüs – Einstellungen] ablehnen. Wenn Sie Automatisch prüfen ablehnen, dann verweigern Sie wirksam die Durchführung aller zukünftiger Softwareaktualisierungen.
Falls Sie Ihr Mobiltelefon bereits verwenden und Sie die Durchführung einer Softwareaktualisierung ablehnen möchten, beachten Sie bitte, dass eine nicht stattgefundene Installation der aktualisierten Software zu einer [Verschlechterung der Leistung Ihres Mobiltelefons, Kompatibilitätsproblemen mit neuen oder vorhandenen Apps und anderen Problemen mit Funktionen oder mit der Sicherheit Ihres Mobiltelefons führen kann].

b.	Abbrechen der Softwareaktualisierung...
Sollten es die Umstände erforderlich machen, können wir unabhängig von den aktuellen Geräteeinstellungen eine Softwareaktualisierung (eine „dringende Softwareaktualisierung“) entsprechend unserer berechtigten Interessen durchführen, ohne dass eine individuelle Benachrichtigung oder die Einholung einer Einwilligung bereitgestellt wird. Zu Beispielen, wann eine dringende Softwareaktualisierung erforderlich sein kann, gehören Bedrohungen für die Sicherheit der Kunden oder die Datensicherheit oder betriebliche Probleme, die zu einer erheblichen Unterbrechung der Dienste oder zu einer unmittelbaren Wahrscheinlichkeit solcher Umstände führen können.

3.	Informationen, die LGE von Ihnen erhalten kann
Beim Durchführen einer Softwareaktualisierung kann LGE die folgenden Informationen über Ihr Mobiltelefon erfassen: Internationale Mobilfunkgerätekennung  (IMEI), Seriennummer, Mobiltelefonmodell, Software- und / oder Firmware-Versionen, Telefonbetriebssystem, Mobiler Ländercode (MCC), Mobilfunknetzcode (MNC), Google Cloud Messaging (GCM) -Kennung und ausgewählte Sprache. Diese Informationen werden im Rahmen der Durchführung einer Softwareaktualisierung automatisch erfasst.

Im Rahmen unserer Aufgabe für eine Bereitstellung von Mobilfunkdiensten für Ihr Mobiltelefon erfassen wir Betriebsinformationen, einschließlich Systemprotokolle mit IP-Adressen. Diese Information wird automatisch erfasst und stellt eine notwendige Voraussetzung für die Bedienung des Telefons dar.

-	Einer Analyse, Prüfung und Verbesserung der Produkte und Dienste, die wir anbieten.

Die rechtliche Grundlage, auf der sich LGE bei der Verarbeitung Ihrer personenbezogenen Daten stützt, ist in der Regel eine der folgenden;
